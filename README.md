# 🌌 ChromaticGravity
### *A living nebula painted by gravity, color, and chaos*

---

## Description

ChromaticGravity is a mesmerizing particle universe where hundreds of color-coded particles orbit dynamically placed gravity wells that you drag freely across the canvas. As particles accelerate, slingshot, and fall into resonant orbits, their colors shift in real time based on velocity and proximity to gravity sources — painting the screen like a cosmic oil-painting that never repeats itself. The result is an endlessly evolving, interactive visualization that lives somewhere between physics simulation and generative art.

---

## ✨ Features

- 🪐 **Draggable gravity wells** — place and move gravity sources anywhere on the canvas to sculpt orbital paths on the fly
- 🎨 **Velocity-reactive color shifting** — each particle's hue responds dynamically to its speed and distance from gravity sources
- ☄️ **Slingshot & resonance effects** — emergent orbital mechanics produce natural-looking slingshots and multi-body resonances
- 🌠 **Chromatic trail painting** — fading trails accumulate into stunning nebula-like compositions over time
- 🔁 **Edge wrapping** — particles seamlessly wrap around canvas boundaries, keeping the universe alive and full
- ⚡ **Smooth 60fps rendering** — velocity capping and damping keep the simulation stable and visually fluid
- 🌐 **Zero-dependency frontend** — pure vanilla JS and the Canvas API, no frontend frameworks required

## 🚀 Installation & Running

> **Prerequisites:** [Node.js](https://nodejs.org/) v14 or higher

```bash
# 1. Clone the repository
git clone https://github.com/sami-23/ChromaticGravity.git

# 2. Navigate into the project directory
cd ChromaticGravity

# 3. Install dependencies
npm install

# 4. Start the server
npm start
```

Then open your browser and visit:

```
http://localhost:3000
```

## 🛠️ Tech Stack

| Layer | Technology | Purpose |
|---|---|---|
| **Runtime** | Node.js | JavaScript server environment |
| **Server** | Express.js `^4.18.2` | Serves static frontend assets |
| **Rendering** | HTML5 Canvas API | Real-time 2D particle rendering |
| **Frontend** | Vanilla JavaScript | Simulation logic, physics, interaction |
| **Styling** | Inline CSS | Fullscreen, overflow-hidden canvas layout |
